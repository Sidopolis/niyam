# Example: Next.js SaaS Project

This example shows a typical Niyam configuration for a Next.js SaaS application.

## Stack

- **Next.js** with TypeScript for the framework
- **Tailwind CSS** for styling
- **Prisma** for database ORM
- **shadcn/ui** for component library

## Usage

```bash
cd your-nextjs-project
niyam init --template startup-saas --tool kiro
```

This generates an `AGENTS.md` (or equivalent) tailored to your full-stack Next.js workflow with clean code, TDD, and performance principles baked in.
