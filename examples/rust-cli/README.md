# Example: Rust CLI Tool

This example shows a minimal Niyam configuration for a Rust command-line application.

## Stack

- **Rust** — systems programming with zero-cost abstractions

## Focus

Clean code and performance principles guide the AI toward idiomatic Rust: proper error handling with `Result`/`?`, zero-copy patterns, and efficient memory usage.

## Usage

```bash
cd your-rust-project
niyam init --tool cursor
niyam add --principle performance
```

This generates a `.cursorrules` file optimized for Rust development with emphasis on ownership patterns, lifetimes, and performant code.
